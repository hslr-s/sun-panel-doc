## 关于开发版必读说明
::: danger 必读！必读！必读！
- v2目前正在开发测试阶段，可能会出现一些不可控的BUG，如果追求稳定的小伙伴，**对 AI 、 docker 和 NAS 相关是刚入门的小伙伴**，**不推荐尝试体验v2开发版本**。
- 前几个开发版本，暂时不兼容 V1 版本的数据备份/迁移，仅支持v1的图标导出配置导入至v2版本（不支持自己上传的图标文件），后面版本会尽快适配，保证让v1的数据迁移至v2版本使用。
- 测试版本（Beta版本）发布后，开发版本将仅支持离线使用，不能登录 Sun-Panel PRO 账号，所以请不要应用在正式环境中。
- 因为想赶在农历新年前发布一个体验版，让大家能都预先体验一些V2的功能和让喜爱开发的小伙伴了解Sun-Panel微应用开发，所以少部分v1的功能在前几个版本还没来得及适配。
:::

## V2 功能
:::tip
以下列出相比v1现已完成的功能项，图标卡片在保留v1版本样式的同时，同组支持更多尺寸的卡片，当然远不止这些，简单挑了一些，有新的建议可以提！
:::
 
- ✅ 增加 微应用/小部件卡片框架，在有限的屏幕里，提升无限的可玩性，灵活安装和卸载。喜爱开发的小伙伴可以按照规范自由开发。
- ✅ 全新的网络环境检测方案，可根据当前访问页面地址，匹配到合适的内网地址、自定义地址内的同网段地址。
- ✅ 增加 卡片自定义地址，可以参与网络环境识别智能打开或右键菜单打开。
- ✅ 重构 导航页面，
    - 跨分组拖拽卡片
    - 增加不同尺寸混用（目前支持【比例（高x宽）】`1x1`,`1x2`,`2x1`,`2x2`,`2x4`,`1x屏宽`），微应用开发者可参照此配置开发不同尺寸的小部件。
    - 手机端完美适配小部件以及其他尺寸卡片
    - 精简卡片dom
- ✅ docker 管理器 从原列表改为卡片风格，显示效果更好
- ✅ 全新docker卡片，2x2卡片支持显示容器的实时网速和累计流量。
- ✅ 图标卡片支持备注文本，仅登录可见。
- ✅ 二进制文件，增加 Mac 端（arm64,amd64）二进制文件运行，取消了cgo的sqlite库支持，采用全go sqlite库（所有端二进制未完全测试，有测试的小伙伴给个反馈结果，感谢）
- ✅ 导入v1图标配置数据
- [ ] 对接三方自托管图标库
- [ ] 内置功能（docker，系统监控等）小部件
- [ ] 侧栏目录
- [ ] v1版本的系统状态组件适配
- [ ] 图库批量删除功能
- [ ] 更多三方适配微应用/小部件
- [ ] 备份功能适配
- [ ] 更多请期待...

## 安装方式

::: tip
与 [v1安装方式](/zh_cn/usage/quick_deploy) 一致。**此为开发版本，目前与v1版本数据不兼容，请勿覆盖更新和应用在正式环境内**
:::

- 首个开发版本拉取命令 `docker pull hslr/sun-panel:2.0.0-dev-1`
    此处文档更新可能不及时，docker 标签名可在v2测试群里公告看到。或参考docker官网：https://hub.docker.com/r/hslr/sun-panel/tags
- 二进制文件下载地址：[夸克盘](https://pan.quark.cn/s/68492917cc31)
