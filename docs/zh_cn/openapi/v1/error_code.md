# 错误码说明

## 公共的

| code |      msg        | 说明|
|--------|----------------------|---|
| 0   | ok           |  无错误 |
| 1100   | token expires    |  服务器错误 |
| 1000   | param format error  |  令牌过期 |
| 1001   | param format error |  参数格式错误 |


## 个别接口会出现的错误
| code |      msg        | 说明|
|--------|----------------------|---|
| 1202   |     only name existed       |  唯一名称已存在 |
| 1203   |     no record           |  无记录 |
