## 非升级安装 `v1.4.0-beta24-02-20+`

如果您是全新安装，你的自定义目录Docker版本：`./conf/custom`，二进制版本：`./custom`

具体参考，没有可以自行创建相关文件
```
├── custom
│   ├── ...
│   ├── index.css // [!code focus]
│   └── index.js // [!code focus]
```
